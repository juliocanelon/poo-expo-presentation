## Spring Framework y Spring Boot

En esta sesión exploraremos:
1. La esencia de Spring Boot y cómo difiere del Spring Framework tradicional.<!-- .element: class="fragment fade-up" -->
2. La forma más rápida de poner en marcha un proyecto y estructurar nuestro código.<!-- .element: class="fragment fade-up" -->
3. La creación de una API REST básica con ejemplos reales.<!-- .element: class="fragment fade-up" -->
4. Funcionalidades avanzadas como Actuator y Spring Data JPA para salud, métricas y acceso a datos.<!-- .element: class="fragment fade-up" -->
5. El ciclo de vida de los beans en Spring.<!-- .element: class="fragment fade-up" -->
6. Finalmente, veremos cómo integrar capacidades de inteligencia artificial usando librerías como openai-java.<!-- .element: class="fragment fade-up" -->
